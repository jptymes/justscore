v4.19 April 10, 2026 Remote Sync — Live Scoreboard on Any Device
New Remote sync — Share your scoreboard live with any device, anywhere. Controller generates a 4-digit code; displays enter the code to connect. Works across networks — phone on Wi-Fi, TV on a separate network, grandma in another state. All see the same live score.
New Display mode — Connected display devices show the scoreboard only. No controls, no menu. Clean full-screen view for a TV or laptop.
New Code display on game clock — When starting a sync session, the 4-digit game code appears on the game clock LED in amber. The shot clock counts down the 60-second join window. All controls are locked during setup except ■ Stop to close the window early.
New Cloudflare-powered relay — Sync is handled by a Cloudflare Worker, not peer-to-peer. No shared network required. Secure: rate limiting prevents code guessing, and sessions close the moment the controller disconnects.
Update Help page updated — Sync controls fully documented in the Controls section at justscore.justintymesports.net/help.

New Game clock adjustment — Four buttons appear below the clock controls when the clock is stopped: −1s, −15s, +15s, +1s. Hidden automatically when the clock is running. Useful for mid-game corrections.
Design ® superscript — JUSTINTYMESPORTS® and JustInTymeSports® now render ® as a proper superscript throughout the app and support pages.
